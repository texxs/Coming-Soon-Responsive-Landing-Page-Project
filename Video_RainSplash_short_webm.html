Rain Splash - Short

Default font set, .webm video format, without fallbacks

HTML 5 best practices require different formats of each video to be included for different operating system/browser/device combinations, as fallbacks in case the first format isn't supported. Only one that will be played will actually be loaded. Almost all devices/browsers can handle all of them and will use the one listed first in the webpage's code. Below you can see how these file formats effect quality and file size (which = loading time).
